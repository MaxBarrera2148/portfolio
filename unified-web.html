Unified Web Ecosystem & Developer Portal

Sr. Webmaster & SEO · RootstockLabs · 2021–2025

Gatsby / React WordPress Docs UX Technical SEO Accessibility

The problem

The company had multiple websites: marketing, docs, community, institutional. Each lived on a different stack and design system. This made updates slow, introduced branding drift, and hurt discoverability in search.

What I did

  • Rebuilt key pages in a shared component system so design and tone stayed consistent.
  • Improved internal routing and navigation across marketing, docs, and institutional content.
  • Refined metadata, IA, and linking to support long-tail developer and institutional search intent.
  • Led relaunch of the developer portal, focusing on clarity, accessibility, and trust.

Why it mattered

Before this work, each team was publishing separately. After, we had one system, not four disconnected sites. Marketing could ship faster. Docs felt more official. Search engines understood how everything fit together.

Result

The developer portal became a core entry point for builders, not just a reference dump. The relaunch was later recognized for accessible multimodal documentation.
